# Brawl Stars Stats v2.0

Инструмент для анализа игроков Brawl Stars с расчётом **Player Skill Index (PSI)**.

## Возможности

- 🔄 Конвертация тегов игроков и кланов (Supercell ID)
- 📊 Полная информация об игроке через Brawl Stars API
- 🧮 Расчёт PSI — система оценки скилла по 7 модулям
- 🌍 Поиск игрока в топ-200 мира и России по бравлерам
- 👥 Информация о клубе и участниках
