# Brawl Stars Stats — Монорепозиторий

Многофункциональный инструмент для анализа игроков Brawl Stars.

## 🏆 Возможности

### 📊 PSI (Player Skill Index)
Уникальная система оценки скилла игрока по 7 модулям:
- **Праймы** — Prestige 1/2/3 с коэффициентами по тирам
- **Ranked** — текущий и максимальный ранг
- **Топ-мир** — позиции в мировых и региональных рейтингах
- **Ветеран** — старые ранги 35 (Legacy)
- **Количественные** — винрейт, трофеи, уровень, плотность праймов
- **Качественные** — серия побед, D/F P3, универсальность
- **Клуб** — участие в клубе

### 🌐 Веб-сайт (Flask + Bootstrap 5)
- Поиск игрока с полным профилем и графиками
- Радар PSI, распределение бравлеров по тирам
- Сравнение игроков
- Рейтинги топ-50 мира и России
- Список бравлеров с тирами и классами
- Мини-игра "Кто круче?"
- Конвертер тегов (игрок/команда)
- Тёмная/светлая тема
- PWA (установка на телефон)
- Калькулятор "Что если?"
- Личный кабинет с историей и избранным
- Бейджи и достижения

### 🤖 Telegram-бот (aiogram)
- `/psi` — расчёт PSI
- `/player` — профиль игрока
- `/card` — карточка с PSI
- `/chart` — график PSI
- `/club` — состав клуба
- `/compare` — сравнение игроков
- `/duel` — дуэль PSI
- `/top` — топ-10 PSI
- `/favorite` — избранное
- `/friend` — друзья (добавить/список/сравнить/активность)
- `/subscribe` — ежедневный отчёт
- `/brawlers` — тиры бравлеров
- `/rankings` — топ-10 мира/России
- `/random` — случайный игрок
- `/events` — ротация карт
- Inline-режим: `@BllsBrawlbot #тег`
- WebApp с графиками
- Развлекательные команды

### 🔧 Инструменты
- Конвертер тегов Supercell ID
- Парсер тиров из Ninja
- Полная информация об игроке (CLI)
- API-клиент Brawl Stars
